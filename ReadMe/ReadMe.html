Read Me Information

This is a collection of fictional hacking tools for use in Live Roleplay Games.

This particular instance is a static hosted version of all the various tools that are located at https://github.com/GandalfsLair/fictional-tools_as_HackerTypers.

This particular repo contains those files, but is linked by a simple index.html file allowing the various tools to be centrally accessed, and accessed via a github static page.

For more information about GitHub static pages see :- https://www.youtube.com/watch?v=5XhxR9Vs6zc&t=123s&pp=ygUVZ2l0aHViIHBhZ2VzIHR1dG9yaWFs - for a introduction.
